The training mode¶

Last layer of abstraction, the coordination: the main loop

bin.akibo.training_mode(max_pop, iterations, graphs, verbose, p1_gen, p2_gen)[source]¶

Runs the training mode

Parameters:
  • max_pop – the maximum size of the population
  • iterations – the maximum number of generations
  • graphs – if we must display graphs on the fly or not
  • verbose – if we must display generation progression or not in the console
  • p1_gen – if not empty, contains a custom first parent genome
  • p2_gen – if not empty, contains a custom second parent genome
